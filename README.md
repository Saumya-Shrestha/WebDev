# WebDev

This repository contains various web development projects created during my 4th semester training. Each project is organized into its own branch for better management and version control.

## Branches

- **main**: Contains the base setup and documentation.
- **CSS-Foundations**: Covers the basics of CSS, including selectors, properties, and styling techniques.
- **Flexbox**: Focuses on CSS Flexbox layout techniques for responsive design.
- **HTML-Foundations**: Introduces the fundamentals of HTML, including structure, elements, and attributes.
- **JS-Fundamentals**: Explores JavaScript basics, including variables, functions, and control structures.
- **Our-Services**: A project showcasing a services page layout.
- **PHP-Fundamentals**: Covers the basics of PHP for server-side scripting.
- **QR-Code**: A project to generate and display QR codes.
- **Recipe**: A project for creating and displaying a recipe page.
- **calculator_app**: A simple calculator application built with JavaScript.
- **document_object_model**: Demonstrates DOM manipulation techniques using JavaScript.

## How to Use

1.  Clone the repository:

    ```sh
    git clone https://github.com/your-username/WebDev.git
    ```

2.  Navigate to the desired branch:
    ```sh
    git checkout branch-name
    ```
